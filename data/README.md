# Sample Data for NLDL Multimodal Tutorial

This folder contains sample datasets and resources for the multimodal learning tutorial.

## Contents

- **sample_text_data.txt**: Example text data for text processing exercises
- **image_labels.csv**: Sample image classification labels
- **dataset_info.txt**: Information about datasets used in the tutorial

## Usage

The notebooks reference this data folder for loading sample datasets. You can replace these files with your own data for experimentation.

## Data Sources

The sample data provided here is synthetic and generated for educational purposes. For real-world applications, consider using:

- COCO Dataset (images + captions)
- Visual Genome (images + scene graphs + Q&A)
- MS-MARCO (text + images)
- Conceptual Captions (images + captions)
